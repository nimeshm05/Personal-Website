Blog Application

Type of project:

Academic-related Database Management System project.

Code:

Codebase

Overview:

The purpose of the blog application system is to facilitate a communication platform. The application system enables users to create, update, and delete their blog posts, create and manage profiles, and raise polls accordingly. This project focuses on implementing the models(database creation and operations) and business logic for the blog application.

Technology Stack:

  • Django framework (python-based web framework) - Used as backend tool to implement and handle business logic, requests, and responses.

  • HTML - Used for organizing content structure on the front-end.

  • Bootstrap - Used for providing styles to the content structure.

  • Heroku - Used as a hosting platform.
