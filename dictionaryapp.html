Dictionary Application

Type of project:

Hackathon Project

Code:

Codebase
Working sample

Overview:

The dictionary app project focuses on web scraping techniques. We scrape the data from two websites to display the word of the day and the phrase of the day. We scrape using the BeautifulSoup module in python. The users can also search to get the definition of words.

Technology Stack:

  • Django framework (python-based web framework) - Used as backend tool to implement and handle business logic, requests, and responses.

  • HTML - Used for organizing content structure on the front-end.

  • Bootstrap - Used for providing styles to the content structure.

  • Heroku - Used as a hosting platform.
