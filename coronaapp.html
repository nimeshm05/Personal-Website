Corona Virus Tracker

Type of project:

College level hackathon project.

Code:

Codebase
Working sample

Overview:

The covid-19 tracker project retrieves statistical numeric information about the coronavirus.
This project focuses on establishing API calls with a live server on the internet and outputting the response data. The details are also listed in the pie chart depending.

Technology Stack:

  • Django framework (python-based web framework) - Used as backend tool to implement and handle business logic, requests, and responses.

  • HTML - Used for organizing content structure on the front-end.

  • CSS - Used for providing styles to the content structure.

  • Heroku - Used as a hosting platform.
